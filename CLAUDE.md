# Stratus Reviews

Code quality and security review tools for GTP-Services teams.

## Available Skills

| Skill | What it does |
|-------|-------------|
| `/review-code` | Run a code quality check on your project |
| `/security-check` | Run a security review on your project |

## How It Works

Each skill spawns a fresh reviewer agent that analyzes your code and reports findings in plain language. Critical issues are auto-fixed. Remaining findings are presented with options to fix, skip, or escalate to the Platform team.
